🛠️ Sobre o Projeto
Este é um script de automação em Bash projetado para transformar uma instalação limpa do Linux Mint/Ubuntu em uma estação de trabalho e jogos completa.
O que este script faz:

    Gaming: Instala Steam, Lutris, Heroic, MangoHud e GameMode.
    Multimídia: Instala OBS Studio, VLC e codecs essenciais.
    Terminal: Configura Zsh, Oh My Zsh e Plugins.
    Apps: Discord e Mission Center.
