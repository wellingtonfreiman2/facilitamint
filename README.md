# facilitamint
Script Bash para setup completo no Linux (Debian/Ubuntu). Instala ferramentas Gamer (Steam, Lutris, Heroic, MangoHud), Design (GIMP + PhotoGIMP, OBS) e Sistema (VLC, Discord). Configura Oh My Zsh com plugins de realce e sugestão, além de instalar Nerd Fonts para um terminal moderno e produtivo. 🚀
Instale pelo terminalcurl -sSL https://githubusercontent.com | bash
